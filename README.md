## magic-tools

我的小工具集合。




## 目录结构

```bash
─ packages                       
    ├─ preconnect-cli             # 获取preconnect链接的脚手架
    ├─ func-line-cli              # 获取源代码所在文件/行的脚手架
    ├─ v-func-line                # VuePress插件，为文档增加指向源代码链接
    ├─ encrypt-tool               # 加解密工具
    ├─ game-2048                  # 2048游戏
    ├─ game-puzzle                # 拼图游戏
    ├─ game-solitaire             # 蜘蛛纸牌游戏
    ├─ game-whiteblock            # 别踩白块游戏
    ├─ game-lantern               # 点灯笼游戏
    ├─ map-service                # 地图服务
    └─ my-docs                    # 一些文档
```
